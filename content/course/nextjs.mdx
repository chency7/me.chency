# Next.js

## 介绍

Next.js 是一个基于 React 的框架，用于构建静态和动态网站。它提供了服务器端渲染（SSR）和静态站点生成（SSG）的功能，使得开发者可以轻松创建高性能的 Web 应用。

## 背景

Next.js 由 Vercel 开发，旨在简化 React 应用的开发过程。它支持自动代码分割、路由、API 路由等功能，极大地提高了开发效率。
